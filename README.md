# Developers Function

運営・開発者向けの多機能ユーティリティプラグイン

概要

Developers Function（DF）は、Minecraft Java Edition向けに開発された運営・開発者支援プラグインです。

サーバー管理、イベント運営、ラグ調査、プレイヤー監視、建築補助など、運営でよく使う機能を1つのプラグインにまとめています。

⸻

対応環境

* Minecraft Java Edition 26.2
* Paper 26.2
  
⸻

主な機能

サーバー管理

* TPS・MSPT確認
* サーバー診断
* アイテム一括削除
* Ping確認
* エンティティ数確認
* プロファイラー

プレイヤー管理

* プレイヤー座標確認
* インベントリ閲覧
* プレイヤー観戦
* 死亡履歴確認

イベント運営

* ボスバータイマー
* カウントダウン
* 全体アナウンス
* チャットロック
* デスカウンター

建築補助

* 湧き潰しチェック
* バイオーム確認
* チャンク情報確認
* 距離測定

ワールド情報

* ワールド情報表示
* 読み込みチャンク確認
* 周辺エンティティ確認

⸻

コマンド

メインコマンド

/df

ヘルプ

/df help

コマンドはカテゴリごとに整理されており、ヘルプから確認できます。

⸻

権限

DF管理者

MinecraftのOPとは別に、DF専用管理者システムを搭載しています。

/df op add <MCID>
/df op delete <MCID>
/df op list

DF管理者は全てのDFコマンドを利用できます。

⸻
